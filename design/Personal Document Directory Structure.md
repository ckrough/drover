# Personal Document Directory Structure

A normalized taxonomy for organizing household and personal files, following NARA-compliant naming conventions.

## Naming Conventions

- Use lowercase alphanumeric characters only
- Use underscores to separate words within a component
- Use hyphens to separate logical components
- Place dates at end in YYYYMMDD format when applicable
- Use descriptive, meaningful names

## Category Definitions

- **career**: Employment history, job applications, professional materials, consulting engagements
- **finance**: Banking, investments, taxes, receipts, insurance, credit
- **food**: Recipes, meal planning, dietary reference
- **healthcare**: Medical records organized by household member and specialty
- **household**: Vehicle records, home systems, emergency planning, maintenance
- **housing**: Property records, leases, real estate transactions
- **legal**: Contracts, identification documents, estate planning
- **lifestyle**: Personal interests, hobbies, travel, volunteering
- **pets**: Animal records, veterinary documents, registrations
- **reference**: General reference materials not belonging to other categories
